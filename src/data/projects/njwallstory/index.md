## Project Overview

NJWALLSTORY is a marketing website built for a wallpaper installation business. The site focuses on showcasing services, projects, and providing a simple way for potential customers to get in touch through a secure contact form.

The project emphasizes **fast performance, responsive design, SEO optimization**, and **secure form submissions** without a full backend server.

## Key Features

- **Marketing Website** — Designed to showcase wallpaper installation services and portfolio.
- **Contact Form** — Users can easily send inquiries; protected with hCaptcha to prevent spam.
- **Serverless Edge Function** — Edge-side handling of contact form submissions via BunnyCDN Edge Script.
- **Responsive Design** — Mobile-first layout built with Tailwind CSS.
- **SEO Optimized** — Structured metadata and accessible components for better search visibility.

## Technology Stack

- **Astro** — Static site generation with partial hydration for interactive components
- **TypeScript** — Type-safe development
- **Tailwind CSS** — Utility-first styling for fast UI development
- **BunnyCDN Edge Script** — Serverless edge functions and caching
- **hCaptcha** — Bot protection for contact forms
- **Vanilla JavaScript / Fetch API** — Client-side form submission handling

- Configured **BunnyCDN Edge Script for serverless form submissions**
