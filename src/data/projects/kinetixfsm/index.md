## Project Overview

KinetixFSM is a modern marketing website developed for a fitness gym to promote memberships, showcase classes and trainers, and provide a smooth online checkout experience for customers.

The project focuses on performance, responsive design, and SEO optimization while enabling secure online payments for gym memberships and services.

## Key Features

- **Modern Marketing Website** — Designed to promote gym memberships, trainers, and services.
- **Online Membership Checkout** — Secure payment flow powered by CloverAPI.
- **Edge Performance Optimization** — BunnyCDN Edge Scripts used for caching and lightweight edge logic.
- **Responsive UI** — Mobile-first layout built with Tailwind CSS.
- **SEO Optimized** — Structured metadata, sitemap, and accessible components for better search visibility.
- **Bot Protection** — Integrated hCaptcha for form security.

## Technology Stack

- **Astro** — Static site generation with partial hydration for interactive components
- **TypeScript** — Type-safe development
- **Tailwind CSS** — Utility-first styling for fast UI development
- **BunnyCDN Edge Script** — CDN edge caching and request handling
- **Clover API** — Payment processing for membership purchases
- **Vanilla JavaScript / Fetch API** — Client-side interactions

## My Role

- Developed the website using **Astro and TypeScript**
- Implemented the **responsive UI using Tailwind CSS**
- Integrated **CloverAPI payment flow**
- Configured **BunnyCDN edge scripts for performance and caching**
- Implemented **security protections using hCaptcha**
