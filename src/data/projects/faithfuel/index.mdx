## 🛠 Technologies Behind the Project

The **FaithFuel** platform was built using a **modern Jamstack architecture** optimized for performance, scalability, and content-first publishing:

- **Astro** – Powers the static site generation (SSG) pipeline for blazing-fast, SEO-optimized pages with minimal JavaScript.
- **Preact** – Provides lightweight reactive UI components (signals) for interactive elements without the React bundle overhead.
- **TypeScript** – Enforces strong typing across the codebase for safer, more maintainable code.
- **TailwindCSS v4** – Delivers a clean, responsive, and utility-first design system for a polished faith-inspired aesthetic.
- **MDX** – Enables rich, component-enhanced content authoring for devotional blog posts.
- **Cloudflare Pages** – Hosts the site on a global CDN with zero cold starts and automatic deployments via Wrangler.
- **Pages CMS** – Provides a Git-based content management interface for managing blog posts and product data without a traditional backend.

---

## 🌟 Key Features

1. **Faith-Inspired Apparel Store**

   - A curated product catalog featuring **Christian-themed t-shirts and clothing** with messages of faith.
   - Products include sizes, stock status, tags, and category filtering for a smooth shopping experience.
   - Clean product detail pages with **optimized image galleries** for each item.

2. **Devotional Blog with MDX**

   - A spiritual growth blog powered by **MDX**, supporting rich content with embedded components.
   - Articles cover topics like faith, God's love, scripture reflection, and Christian living.
   - Structured with **featured posts**, tags, keywords, and article sections for SEO discoverability.

3. **Git-Based Content Management with Pages CMS**

   - Blog posts and products are managed through **Pages CMS**, enabling non-technical contributors to publish content directly via a user-friendly UI backed by Git.
   - No database required — content lives in the repository as **Markdown and JSON files**.

4. **SEO & Performance Optimized**

   - **Schema.org structured data** and a generated sitemap improve search engine visibility.
   - **Dynamic OG image generation** (via Satori and Sharp) creates custom social share images per page.
   - Astro's static output means **zero server-side latency** — every page is pre-rendered HTML.

5. **Cloudflare-Powered Global Delivery**
   - Deployed on **Cloudflare Pages** for instant global distribution across 300+ edge locations.
   - Custom deployment pipeline via **Wrangler CLI** for streamlined production releases.
